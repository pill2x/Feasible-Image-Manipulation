# Feasible-Image-Manipulation Python Tool
Elevate Your Visuals with Ease!

This project empowers you to manipulate images using Python. From resizing to applying filters, enhance your creativity with a range of transformations.

## Features

- Resize image
- Apply filters for artistic effects
- Simple yet powerful transformations

## Getting Started

1. Clone the repository: `git clone https://github.com/pill2x/image-manipulation.git`
2. Install required packages: `pip install -r requirements.txt`
3. Run the image manipulation script: `imageManipulation.py`

## Usage

- Place the images you want to manipulate in the `images` folder.
- Edit the 'image_path' directory 'C:/Users/HP/OneDrive/Desktop/Computer Vision/images/Catfie.jpg' with desired user directory.
- Run the script to see the transformed images.
